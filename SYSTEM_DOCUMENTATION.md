## ğŸ—ï¸ System Overview

The **Purok Management System (PMS)** is a comprehensive web-based application designed to manage youth activities, events, announcements, and attendance tracking for the Sangguniang Kabataan (SK) organization using QR code technology.

**System Type:** Web Application (PHP-based)  
**Architecture:** MVC-like Pattern  
**Database:** MySQL (MariaDB)  
**Framework/Libraries:** AdminLTE, Bootstrap 4, jQuery, Chart.js  
**Server:** Apache (XAMPP)

## ğŸ› Troubleshooting

### Common Issues

**Issue: QR Scanner not working**
```
Solution:
- Check browser permissions (camera access)
- Use HTTPS or localhost
- Clear browser cache
- Check JavaScript console for errors
```

**Issue: Images not uploading**
```
Solution:
- Check uploads/ folder permissions (777)
- Verify PHP file upload settings in php.ini
  - upload_max_filesize = 10M
  - post_max_size = 10M
- Check disk space
```

**Issue: Database connection error**
```
Solution:
- Verify MySQL is running
- Check database credentials in initialize.php
- Ensure database exists
- Test connection using phpMyAdmin
```
