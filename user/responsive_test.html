Responsive Breakpoints Guide

Breakpoint Screen Width Key Changes
Mobile < 768px Single column, mobile menu, adjusted padding
Tablet 768px - 992px 2 columns, mobile menu, medium padding
Laptop (Small) 993px - 1199px Compact nav, smaller fonts, 2-3 columns

Desktop > 1366px Full layout, max-width 1400px

Testing Instructions

  3. Verify the breakpoint indicator at the bottom-right shows correct breakpoint
  4. Test the mobile menu by resizing below 993px
  5. Scroll horizontally - there should be NO horizontal scrollbar
  6. Check footer - text should wrap properly and not overflow
